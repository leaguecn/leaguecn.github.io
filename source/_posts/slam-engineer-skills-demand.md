## 学习要求
### 技能：
+ 【熟练使用C和C++】

    + 《21天学通C++》
    + 《Effective C++》

+ 【熟悉常用的opencv，等库】
    + 《opencv 3入门》

    + 《学习 opencv》
+ 【熟悉linux下编程】

    + 熟悉cmake编程
    + 熟悉Kdevelop 以及调试

+ 【QT下的编程】
    + 熟悉QT软件以及基本的语法功能

+ 【熟悉常见的slam框架】
    + 跟着高翔的14讲走就行

+ 【机器学习数学知识】
    + 高数，矩阵分析，概率论

+ 【基本的数据结构与算法】
    + 《数据结构，邓俊辉》书籍

    + 《算法竞赛入门》睡觉
+ 【熟悉模式识别，多视图几何，计算机视觉等理论知识】

    + 《计算机视觉》书籍
    + 《多视图几何》某几章

    + 《模式识别》书籍
### 项目经历：
